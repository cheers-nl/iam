# Identity Federation Pain Log

Friction observations captured while setting up workforce federation from Microsoft Entra ID into AWS Console via IAM Identity Center. Format matches [`../team-vault/pain-log.md`](../team-vault/pain-log.md) so the two exercises stack into one consistent observation dataset.

Severity rubric (same as Team Vault):
- **low** — annoyance, costs minutes, single-event friction
- **medium** — costs an hour or causes a meaningful detour
- **high** — blocks progress, costs a half-day or more, or is a category mistake (security misconfiguration, lock-out, data loss risk)

Category tags (free-form, but try to reuse): Onboarding / Setup, Naming, Docs, Permissions, Error messages, IAM-IdP boundary, IAM Identity Center, SAML, SCIM, Permission sets, Account assignment, AWS access portal, Console UX.
